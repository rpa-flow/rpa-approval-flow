# Corporate Precision Design System

Este sistema de design foi criado para a **Minas Mineração**, focando em produtividade, confiança e precisão técnica.

## Princípios de Design
1. **Fidelidade à Marca**: Uso do azul profundo (`#2b3a7e`) como cor primária para transmitir autoridade e estabilidade, e o verde água/teal (`#23a18e`) como acento para representar crescimento e modernidade.
2. **Clareza Operacional**: Interfaces limpas com baixa densidade visual para facilitar a tomada de decisão rápida em fluxos de aprovação.
3. **Contraste e Legibilidade**: Uso da fonte Manrope para garantir legibilidade técnica em tabelas e dados numéricos.

## Componentes Principais
- **Cards**: Fundo branco, bordas de 4px e sombras discretas (`shadow-sm`) para separar o conteúdo do fundo cinza-azulado.
- **Status Badges**: Cores semânticas com saturação controlada para evitar fadiga visual.
- **Navegação**: Sidebar com hierarquia clara e ícones lineares.
